# MusicSmell

Εφαρμογή εκτέλεσης πειράματος διατροπικού σημασιολογικού priming (όσφρηση ↔ ακοή).

Τρέχει τοπικά σε browser, δεν χρειάζεται σύνδεση στο internet.

---

## Τι χρειάζεστε

- Έναν υπολογιστή με **Python 3** εγκατεστημένο (Mac: υπάρχει ήδη, Windows: [python.org](https://www.python.org/downloads/))
- Έναν σύγχρονο browser (Chrome ή Firefox)
- Τα αρχεία ήχου `.wav` (5 ήχοι + pink noise)
- Τα CSV αρχεία με τη σειρά των trials (ένα ανά participant/session)

---

## Εκκίνηση

### Mac
Κάντε double-click στο αρχείο **`start.sh`**.

### Windows
Κάντε double-click στο αρχείο **`start.bat`**.

Θα ανοίξει ένα παράθυρο terminal και ο browser αυτόματα. **Μην κλείσετε το terminal** — ο server τρέχει εκεί. Όταν τελειώσετε, κλείστε το terminal.

---

## Βήμα-βήμα χρήση

### 1. Ρύθμιση
Στο browser θα δείτε μια μαύρη οθόνη που γράφει **«Αναμονή...»**. Αυτή είναι η οθόνη του συμμετέχοντα.

Πατήστε το μικρό κουμπί **«Ρύθμιση»** (κάτω δεξιά) — θα ανοίξει ένα δεύτερο παράθυρο, η **οθόνη ερευνητή**.

Σύρετε κάθε παράθυρο στη σωστή οθόνη:
- Οθόνη συμμετέχοντα → στον participant
- Οθόνη ερευνητή → σε εσάς

### 2. Φόρτωση αρχείων (στην οθόνη ερευνητή)
1. Πατήστε **«Επιλέξτε φάκελο ήχων»** → επιλέξτε τον φάκελο `audio/` που περιέχει τα `.wav` αρχεία
2. **Σύρετε** το CSV αρχείο του participant (π.χ. `P01_S1.csv`) στην περιοχή drag & drop, ή κάντε κλικ για να το επιλέξετε
3. Πατήστε **«Έναρξη Πειράματος»**

### 3. Εκτέλεση πειράματος
Ο συμμετέχων βλέπει τα ερεθίσματα αυτόματα. Εσείς βλέπετε στην οθόνη σας:
- Τι trial τρέχει και σε ποια φάση βρίσκεται
- **Πότε να δώσετε άρωμα** (πορτοκαλί ειδοποίηση)
- Τι έρχεται στο επόμενο trial

Όταν εμφανίζεται οσφρητικό ερέθισμα:
- **«Ετοιμάστε: [άρωμα]»** → ετοιμάστε το μπουκαλάκι
- **«ΔΩΣΤΕ: [άρωμα]»** → δώστε το στον συμμετέχοντα

### 4. Τέλος session
Μόλις τελειώσουν τα trials:
- Ο συμμετέχων βλέπει **«Ευχαριστούμε»**
- Κατεβαίνει **αυτόματα** ένα CSV αρχείο αποτελεσμάτων (π.χ. `results_P1_S1.csv`)

### 5. Επόμενος participant
Πατήστε **«Νέο Session»** στην οθόνη ερευνητή. Οι ήχοι παραμένουν φορτωμένοι — σύρετε μόνο το νέο CSV και πατήστε «Έναρξη».

---

## Μορφή αρχείων

### CSV εισόδου
```
ParticipantID,Session,TrialNum,TargetType,Target,PrimeType,Prime,Congruency
1,1,1,Olfactory,Tobacco,Auditory,Pink noise,Control
1,1,2,Olfactory,Black pepper,Auditory,Vanilla.wav,Incongruent
...
```

### CSV εξόδου
Ίδιες στήλες + δύο επιπλέον:
- **Response** — η λέξη που επέλεξε ο συμμετέχων
- **ResponseTimeMs** — χρόνος αντίδρασης σε χιλιοστά δευτερολέπτου

---

## Χρονισμοί

| Φάση | Διάρκεια |
|------|----------|
| Prime (ερέθισμα) | 3 δευτερόλεπτα |
| Παύση μεταξύ prime–target | 1 δευτερόλεπτο |
| Target (ερέθισμα) | 3 δευτερόλεπτα |
| Απόκριση | Χωρίς χρονικό όριο |
| Παύση μεταξύ trials | 1 δευτερόλεπτο |

---

## Σε περίπτωση προβλήματος

- **Ο browser δεν ανοίγει αυτόματα**: πηγαίνετε χειροκίνητα στο `http://localhost:8000`
- **Τα popups μπλοκάρονται**: επιτρέψτε popups στον browser για `localhost`
- **Crash ή κλείσιμο κατά λάθος**: τα αποτελέσματα αποθηκεύονται αυτόματα μετά από κάθε trial στον browser (localStorage)
